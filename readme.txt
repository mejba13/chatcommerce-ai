24/7 AI support and lead capture for WooCommerce with OpenAI-powered chat, product-aware responses, and modern, customizable UI.

== Description ==

ChatCommerce AI brings intelligent, 24/7 customer support to your WooCommerce store. Powered by OpenAI's advanced language models, it provides instant, accurate responses to customer inquiries about products, shipping, returns, and more.

= Key Features =

* **AI-Powered Chat**: OpenAI GPT-4 integration for natural, helpful conversations
* **Product Knowledge**: Automatically syncs your product catalog for accurate product recommendations
* **Real-Time Streaming**: Responses stream in real-time for a smooth user experience
* **Lead Capture**: Collect customer contact information with GDPR-compliant consent
* **Feedback System**: Gather customer satisfaction data with thumbs up/down ratings
* **Customizable UI**: Match your brand with custom colors, positioning, and welcome messages
* **Analytics Dashboard**: Track sessions, leads, and customer satisfaction
* **Privacy-First**: GDPR/CCPA ready with data retention controls

= How It Works =

1. Install and activate the plugin
2. Add your OpenAI API key in settings
3. Customize the chatbot appearance and behavior
4. The AI automatically learns about your products and policies
5. Start providing 24/7 support to your customers!

= Requirements =

* WordPress 6.4 or higher
* WooCommerce 8.0 or higher
* PHP 8.1 or higher
* OpenAI API key (get one at platform.openai.com)

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/chatcommerce-ai/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to ChatCommerce AI → Settings
4. Enter your OpenAI API key
5. Configure your preferences and enable the chatbot

== Frequently Asked Questions ==

= Do I need an OpenAI account? =

Yes, you need an OpenAI API key. Sign up at https://platform.openai.com/ and create an API key.

= How much does OpenAI cost? =

OpenAI charges based on usage (tokens). Costs are typically $0.01-0.03 per conversation. Monitor your usage in your OpenAI dashboard.

= Is the chat widget mobile-friendly? =

Yes! The widget is fully responsive and works beautifully on all devices.

= Can I customize the chatbot's responses? =

Absolutely! Edit the system instructions in Settings → Instructions to define your chatbot's personality and behavior.

= Is my customer data secure? =

Yes. We implement encryption, GDPR compliance, and follow WordPress security best practices. Conversations are processed by OpenAI according to their privacy policy.

= Can I export leads? =

Yes! Export leads to CSV from the Leads page, or set up webhook integrations for your CRM.
